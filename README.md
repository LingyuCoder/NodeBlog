天镶的博客
====
一个基于Node.js开发的公共博客管理系统，使用express，mongodb进行开发

目前已部署到[nodejitsu](https://www.nodejitsu.com/)上，数据库使用[mongohq](http://www.mongohq.com/home)，[单击访问主页](http://skyinlayer.nodejitsu.com/)
##目前的角色
1. 管理员
2. 来访用户
##目前的功能
###系统
1. 普通用户注册
2. 瀑布流查看博客图片
###所有用户
1. 查看/评论文章
2. 对评论点赞
3. 收藏文章
4. 查看用户中心
5. 修改基本的个人信息
###管理员额外功能
1. 添加/修改/删除 文章
2. 添加/删除 图片
##添加中的功能
1. 查看他人用户中心
2. 用户关注
3. 图片点赞
4. 文章访问量统计
5. ...
